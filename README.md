## 環境建置

* 【NPM】 是 Node Package Manager 的簡稱，它是一個線上套件庫，可以下載各式各樣的 Javascript 套件來使用。
* 【ngrok】 一個可以把你 localhost 對應到 https public domain 的服務。例如：把你的 localhost:5000 對應到 https://fj2rijo3.ngrok.com。
